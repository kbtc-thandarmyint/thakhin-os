EASY INSTALLATION GUIDE

လိုအပ်ချက်များ (System Requirements)

  1. PC / Intel Mac (amd64): 64-bit (x86-64) CPU — Windows laptop ဟောင်းသစ်မရွေး၊ Linux PC များ၊ Intel Mac များ။ VirtualBox / VMware / USB boot အားလုံးရ။
  2. Apple Silicon Mac (arm64): M1/M2/M3/M4 — UTM ဖြင့် arm64 ISO ကို native run (နည်းလမ်း ၃ ကို ကြည့်ပါ)။
  3. အနည်းဆုံး RAM 4 GB (security tools အတွက် 6 GB+ အကြံပြု)၊ disk 25–30 GB။

နည်းလမ်း ၁ — VirtualBox တွင် စမ်းသပ်ခြင်း (အလွယ်ဆုံး)

  1. VirtualBox ကို virtualbox.org မှ download လုပ်၍ install လုပ်ပါ။
  2. New နှိပ်ပြီး — Name: Thakhin OS, Type: Linux, Version: Debian (64-bit) ရွေးပါ။
  3. Memory 4096 MB, Virtual disk 25 GB (VDI, dynamically allocated) ပေးပါ။
  4. Settings → Storage → Optical Drive တွင် ThakhinOS-1.0-amd64.iso ကို ချိတ်ပါ။
  5. Start နှိပ်ပါ — ရွှေရောင် boot menu ပေါ်လာလျှင် Thakhin OS 1.0 Live ကို ရွေးပါ။
  6. Desktop ရောက်လျှင် "Install Thakhin OS (သခင် OS တပ်ဆင်ရန်)" icon ကို double-click နှိပ်ပါ။
  7. Installer အဆင့်များအတိုင်း သွားပါ — install နေစဉ် ဥက္ကဋ္ဌကြီး quiz slideshow ကို ခံစားပါ။ ပြီးလျှင် reboot လုပ်ပြီး ISO ကို ဖြုတ်ပါ။

နည်းလမ်း ၂ — USB ဖြင့် Laptop/PC အစစ်တွင် တပ်ဆင်ခြင်း

  1. 8 GB+ USB stick တစ်ချောင်း ပြင်ပါ (ထဲက data အားလုံး ပျက်မည် — backup ဦးစွာလုပ်ပါ)။
  2. Windows တွင်: Rufus (rufus.ie) သို့မဟုတ် balenaEtcher ဖြင့် ISO ကို USB ထဲ ရေးပါ။
    macOS / Linux တွင်: balenaEtcher သို့မဟုတ် sudo dd if=ThakhinOS-1.0-amd64.iso of=/dev/sdX bs=4M status=progress
  3. Laptop ကို restart လုပ်ပြီး boot menu ဝင်ပါ — အများအားဖြင့် F12 (Dell/Lenovo), F9 (HP), F2/Del (Asus/Acer/MSI), Intel Mac တွင် Option (⌥) ကို ဖိထားပါ။
  4. USB ကို ရွေး၍ boot လုပ်ပါ — BIOS ရော UEFI ရော အလုပ်လုပ်သည် (Secure Boot ပိတ်ရန် လိုနိုင်သည်)။
  5. Live desktop ရောက်လျှင် Install icon နှိပ်ပြီး လမ်းညွှန်အတိုင်း ဆက်သွားပါ — Windows နှင့် dual-boot လိုလျှင် installer ၏ "Install alongside" ကို ရွေးပါ။

နည်းလမ်း ၃ — Apple Silicon Mac (M1/M2/M3/M4) — UTM

  1. UTM ကို mac.getutm.app (သို့) App Store မှ install လုပ်ပါ — အခမဲ့။
  2. arm64 ISO (ThakhinOS-1.0-arm64.iso) ကို download လုပ်ပါ — Apple Silicon အတွက် native ဖြစ်၍ အလွန်မြန်သည် (emulation မဟုတ်)။
  3. UTM → Create a New Virtual Machine → Virtualize → Linux ရွေးပါ။
  4. Boot ISO image တွင် ThakhinOS-1.0-arm64.iso ကို ရွေးပါ။ RAM 4 GB+, disk 25 GB+ သတ်မှတ်ပါ။
  5. VM ကို start လုပ်ပါ — GRUB (UEFI) ရွှေရောင် menu မှ Thakhin OS 1.0 Live ကို ရွေးပါ။
  6. Desktop ရောက်လျှင် Install icon နှိပ်၍ လမ်းညွှန်အတိုင်း ဆက်သွားပါ။
  7. amd64 ISO ကို Apple Silicon တွင် UTM ၏ Emulate mode ဖြင့်လည်း run နိုင်သော်လည်း များစွာ နှေးသည် — arm64 ကို အသုံးပြုရန် အကြံပြုသည်။ VirtualBox သည် Apple Silicon တွင် amd64 ISO ကို boot မလုပ်နိုင်ပါ။

တပ်ဆင်ပြီးနောက် ပထမဆုံးလုပ်ရန်

  1. Terminal ဖွင့်ပါ — neofetch နှင့် ဥက္ကဋ္ဌကြီး fact တစ်ခု ကြိုဆိုလိမ့်မည်။
  2. tarzan-check ရိုက်၍ မြို့သားသခင် စစ်ဆေးမှု ဖြေဆိုပါ — ၉ မှတ်အထက်ရလျှင် စစ်မှန်သော သခင်။
  3. thakhin-arsenal ဖြင့် security tools စာရင်းကြည့်ပါ။
  4. System update: thakhin apt update && thakhin apt upgrade

Windows လိုင်စင် လိုသလား? ကုန်ကျစရိတ်က?

ဘာမှမလို — အခမဲ့ (free & open source) ဖြစ်သည်။ ဥက္ကဋ္ဌကြီး၏ စပွန်ဆာကြေးများနှင့် မတူဘဲ ဤ OS မှာ တစ်ပြားမှ မပေးရ။

Updates ဘယ်လိုရမလဲ?

Debian repository များမှ ပုံမှန်အတိုင်း — sudo apt update && sudo apt upgrade။ VS Code မှာလည်း Microsoft repo ချိတ်ပြီးသားဖြစ်၍ အလိုအလျောက် update ရသည်။

sudo ကို thakhin နဲ့ အစားထိုးထားတာလား? sudo ဆက်သုံးလို့ရသေးလား?

Root အခွင့်အာဏာ တောင်းရန် thakhin <command> ကို သုံးသည် (ဥပမာ thakhin apt update)။ မူရင်း sudo binary ကိုမူ ဖျက်မထားပါ — script/package များ ပျက်စီးမသွားစေရန် ဆက်လက်အလုပ်လုပ်နေသည်။ thakhin မှာ sudo ၏ themed wrapper ဖြစ်ပြီး Burmese lecture နှင့် ☸ prompt ပါဝင်သည်။

Security tools တွေက တရားဝင်လား? ဘယ်လိုသုံးရမလဲ?

nmap, Metasploit, Wireshark, aircrack-ng စသည့် tools အားလုံးသည် Kali/Parrot OS တို့နည်းတူ လွတ်လပ်စွာ ရရှိနိုင်သော၊ security professional များ တရားဝင်သုံးသော tools များဖြစ်သည်။ ကိုယ်ပိုင် (သို့) authorized system များ၊ CTF ပြိုင်ပွဲများ၊ ပညာရေး အတွက်သာ သုံးပါ။ အခွင့်မရှိသော စနစ်များကို တိုက်ခိုက်ခြင်းသည် တရားမဝင်ပါ — တာဇံ လုပ်ရပ်ဖြစ်သည်။

amd64 နဲ့ arm64 — ဘယ်ဟာ download လုပ်ရမလဲ?

Windows/Linux PC သို့ Intel Mac သို့ VirtualBox ဆိုလျှင် amd64။ Apple Silicon (M1–M4) Mac ပေါ်တွင် UTM ဖြင့် run မည်ဆိုလျှင် arm64 (native, မြန်သည်)။

တာဇံတစ်ယောက်က ဒီ OS သုံးရင် ဘာဖြစ်မလဲ?

တပ်ဆင်ပြီးသည်နှင့် မြို့သားသခင်အဖြစ် အလိုအလျောက် အသိအမှတ်ပြုခံရသည် — ရှေ့တန်းအမှတ် (၁) စခန်းသို့ သွားစရာ မလိုတော့ပါ။ ပညာပေးအောင်မြင်မှု တစ်ခုအဖြစ် MTC မှ မှတ်တမ်းတင်မည်။
